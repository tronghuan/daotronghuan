# AI API — Build with AI

Ba loại tool trước (Chat, Code Editor, Coding Agent) là dùng AI để **làm việc hiệu quả hơn**. AI API là dùng AI để **xây dựng sản phẩm** — tích hợp khả năng AI vào app của mình.

## Khi nào cần AI API?

| Use case | Ví dụ |
|----------|-------|
| **Chatbot** | Hỗ trợ khách hàng tự động, FAQ bot |
| **Summarization** | Tóm tắt tài liệu dài, email, báo cáo |
| **Extraction** | Rút trích thông tin từ văn bản phi cấu trúc |
| **Classification** | Phân loại ticket support, phân loại sản phẩm |
| **Generation** | Sinh nội dung, mô tả sản phẩm, email template |
| **Translation** | Dịch đa ngôn ngữ tích hợp vào app |
| **Code analysis** | Tự động review, lint, generate docs |

## So sánh các AI API

| | Claude API | OpenAI API | Gemini API |
|---|---|---|---|
| **Model mạnh nhất** | Claude Opus 4.5 | GPT-4o | Gemini 1.5 Pro |
| **Giá (input)** | Cạnh tranh | Cao hơn | Có free tier |
| **Context window** | 200K tokens | 128K | 1M tokens |
| **Tool use** | Tốt | Tốt | Tốt |
| **Đặc điểm** | Safety focus, honest | Phổ biến nhất | Multimodal mạnh |
