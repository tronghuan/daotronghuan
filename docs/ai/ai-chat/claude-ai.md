# Claude.ai cho Developer

> Nội dung đang được chuẩn bị. Quay lại sớm nhé!

## Sắp có trong bài này

### Tại sao Claude phù hợp cho Developer?
- Context window lớn — paste cả file code dài mà không bị cắt
- Lập luận chặt chẽ — giải thích rõ "tại sao" chứ không chỉ "cái gì"
- Honest về giới hạn — nói thẳng khi không biết thay vì bịa
- Code quality cao — tuân theo best practices và clean code

### Claude Projects
- Tạo Project để duy trì context qua nhiều cuộc trò chuyện
- Upload file: codebase, docs, design specs
- Custom instructions cho Project (tương đương CLAUDE.md)
- Use case: một Project riêng cho mỗi dự án đang làm

### Artifacts
- Claude tạo code/document có thể preview trực tiếp
- Chỉnh sửa artifact trong cùng conversation
- Export code ra clipboard

### Tính năng hữu ích cho Developer
- **Code generation**: từ description → code đầy đủ
- **Code explanation**: paste code → giải thích từng dòng
- **Debugging**: mô tả bug → Claude phân tích nguyên nhân
- **Refactoring**: paste code cũ → nhận code đã cải thiện
- **Test generation**: paste function → nhận unit tests
- **Documentation**: paste code → nhận JSDoc/docstring

### Claude.ai vs Claude API vs Claude Code
- **Claude.ai** — giao diện web, dùng tay, tốt cho brainstorm và hỏi đáp
- **Claude API** — tích hợp vào sản phẩm, programmatic
- **Claude Code** — CLI agent, tự động trong terminal

### Tips sử dụng hiệu quả
- Cung cấp context đầy đủ: ngôn ngữ, framework, constraint
- Yêu cầu giải thích khi nhận code — không chỉ copy-paste
- Dùng follow-up: "tại sao dùng approach này?", "có cách nào khác không?"
- Paste error message đầy đủ khi debug
