# AI Chat cho Developer

AI Chat là dạng AI tool **dễ tiếp cận nhất** — chỉ cần gõ câu hỏi và nhận câu trả lời. Nhưng dùng đúng cách, nó có thể là công cụ tư duy mạnh nhất trong arsenal của developer.

## Khi nào dùng AI Chat?

| Use case | Ví dụ |
|----------|-------|
| **Giải thích code** | "Đoạn regex này làm gì?" |
| **Debug bằng mô tả** | "Function này trả về undefined, tôi không hiểu tại sao..." |
| **Brainstorm kiến trúc** | "Tôi cần design một hệ thống cache, có những approach nào?" |
| **Học khái niệm mới** | "Giải thích Event Loop trong Node.js" |
| **Code review** | "Review đoạn code này và chỉ ra vấn đề" |
| **Viết nội dung kỹ thuật** | "Viết commit message, comment, docs cho hàm này" |

## So sánh nhanh

| | Claude (Anthropic) | ChatGPT (OpenAI) | Gemini (Google) |
|---|---|---|---|
| **Điểm mạnh** | Lập luận dài, code chất lượng cao, honest | Phổ biến, plugins, DALL-E | Tích hợp Google, multimodal |
| **Context window** | Rất lớn (200K tokens) | Lớn (128K) | Lớn (1M - Gemini 1.5) |
| **Tốt nhất cho** | Technical depth, long documents | General purpose | Google Workspace integration |
| **Free tier** | Có (Claude.ai) | Có (GPT-3.5) | Có |
