# AI Code Editor

AI Code Editor đưa AI vào **ngay trong môi trường code** — không cần chuyển tab, AI hiểu cả codebase và viết code trực tiếp vào file.

## Điểm khác biệt so với AI Chat

| | AI Chat | AI Code Editor |
|---|---|---|
| **Context** | Bạn phải paste code | AI tự đọc cả codebase |
| **Output** | Text/code cần copy | Viết thẳng vào file |
| **Workflow** | Tách rời khỏi editor | Tích hợp, không rời tay |
| **Tốt cho** | Tư duy, brainstorm | Viết code, refactor |

## Khi nào dùng AI Code Editor?

- **Autocomplete thông minh** — không chỉ gợi ý từ tiếp theo mà cả function hoàn chỉnh
- **Chat trong context codebase** — hỏi về code đang mở, file liên quan
- **Multi-file edit** — "thêm validation cho form này và cập nhật tests tương ứng"
- **Inline edit** — chọn đoạn code, ra lệnh sửa ngay tại chỗ
- **Generate từ comment** — viết comment mô tả, AI viết code
