# Prompt Engineering

Prompt Engineering là **kỹ năng nền tảng** áp dụng cho tất cả 4 loại AI tool. Viết prompt tốt hơn = kết quả tốt hơn, tiết kiệm thời gian hơn.

## Tại sao Prompt Engineering quan trọng?

Cùng một câu hỏi, viết prompt khác nhau cho kết quả khác nhau:

```
❌ Prompt kém:    "Fix bug này"
✅ Prompt tốt:   "Function calculateTax() trả về sai khi input là 0.
                  Expected: 0, Actual: NaN.
                  Nguyên nhân có thể là gì và cách fix?"
```
