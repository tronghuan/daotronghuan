Core ConceptsвҖӢ

JobвҖӢ

Job lГ  gГ¬: Mб»ҷt Job = mб»ҷt pipeline ETL hoГ n chб»үnh. TЖ°ЖЎng tб»ұ mб»ҷt file Python script hay mб»ҷt Workflow trong cГЎc tool khГЎc. Khi bбәЎn "Run" mб»ҷt Job, Talend:

VГӯ dб»Ҙ thб»ұc tбәҝ: Job ImportEmployees Д‘б»Қc file CSV, validate dб»Ҝ liб»Үu, insert vГ o MySQL. Job ExportReports query MySQL, transform, ghi ra Excel.

Mб»—i component cГі:

  • Advanced settings вҖ” config nГўng cao (encoding, buffer size, null handling...)

  • View вҖ” hiб»ғn thб»Ӣ/бә©n component trГӘn canvas
