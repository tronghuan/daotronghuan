# Salesforce

Series tài liệu về Salesforce — từ những kiến thức cơ bản đến kiến trúc nâng cao.

## Các chủ đề

- **Apex** — Ngôn ngữ lập trình của Salesforce
- **LWC** — Lightning Web Components
- **Flow** — Tự động hóa không cần code
- **SOQL** — Salesforce Object Query Language
- **Integration** — Tích hợp Salesforce với hệ thống ngoài
